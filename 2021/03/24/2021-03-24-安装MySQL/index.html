安装MySQL

Created2021-03-24|Updated2021-11-25|MySQL
|Post View:
协同开发要用统一的标准，引起的MySQL之痛

不要问我为什么不能在原有的基础上装，我也没试过:cry:

卸载MySQL

  • 停止MySQL服务
  • 在/bin下 运行mysqld --remove

下载MySQL

  • 一定要确保自己解压后的bin 目录上有mysql.exe，不然怎么执行mysqld 都没用
  • 解压目录随便放，我在D盘

安装MySQL

  • 先改环境变量，删除旧的，添加新的，在系统变量下的Path

    ………………\bin

  • 管理员运行cmd，切换到bin目录下

  • 然后mysqld --initialize --console

    顺便记下最后弹出的root账户密码，一般比较复杂

  • 安装服务

    mysqld --install

  • 运行服务

    net start mysql

  • 登录，修改

    mysql -u root -p

    UPDATE user SET password = PASSWORD('新密码') WHERE user = 'root';

  • 完成
