bolg_remake

Created2021-10-30|Updated2021-10-30|Essays
|Post View:

之前是jekyll+nexT配置的博客，后来看到https://blog.bxhong.cn/，就换用hexo+butterfly了

必要的环境

  • node
sudo apt update
sudo apt upgrade
curl -sL https://deb.nodesource.com/setup_16.x | sudo -E bash -
sudo apt-get install -y nodejs
node -v
  • npm
sudo apt install -y npm
sudo npm install -g n
sudo n stable
npm -v
  • git
sudo apt install git
git -v
  • Hexo
npm install -g hexo-cli
npm install hexo-deployer-git --save

搭建

mkdir ~/web
cd ~/web
hexo init myblog
cd myblog
npm install
git clone -b master https://github.com/jerryc127/hexo-theme-butterfly.git themes/butterfly

_config.yml

deploy:
  type: 'git'
  repo:
    github: git@github.com:AmorFati-coder/AmorFati-coder.github.io.git
  branch: master

详细参数参考Doc

github上传

详细的步骤不说，主要注意两点

  • _config 
    deploy:
  type: 'git'
  repo:
    github: git@github.com:AmorFati-coder/AmorFati-coder.github.io.git //这里一定要写ssh的链接，否则就得频繁输入account和token
  branch: master
  • 本地ssh的生成，github ssh的配置

域名的绑定

  • 云服务商DNS添加CNAME记录
  • touch themes/butterfly/source/CNAME添加域名
  • page选项添加域名

常用命令

  • hexo s本地运行&调试
  • hexo g编译md文件，生成public目录
  • hexo clean删除public目录
  • hexo g -d上传
  • hexo new page 'name'新的子页面
  • hexo new 'name' 新的文章

其他

  • 其实也可以编译html，不过注意作用范围
